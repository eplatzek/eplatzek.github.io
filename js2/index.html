    Coding Challenge
    Layout: 2 Text Inputs, a button to run our JS and an area for results.

  1. First Challenge: Create two text inputs, a div with the class "results", and a button attached to a JS function using onclick.
  2. Second Challenge: When we click the button, we will console log the larger of the two strings (based on string length).
  3. Third Challenge: When we click the button, we will use DOM manipulation to change the "results" div to contain the larger of the two strings.
  4. Fourth Challenge: Our client decided that they want a more complicated behavior. If the text entered is a number then we need to compare the number to the length of the other text. The largest string or biggest number will be displayed in the results area
    Hint: You will need 4 comparisions (number vs number, string vs string, number vs string, string vs number) and a way to determine if the text is a number.
