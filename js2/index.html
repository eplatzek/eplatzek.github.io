    Coding Challenge
    Layout: 2 Text Inputs, a button to run our JS and an area for results.

  1. First Challenge: The desired behavior is that after we have typed in text in both text inputs and clicked the button, we display the larger string of the two inputs in the results section.
  2. Second Challenge: Our client decided that they want a more complicated behavior. If the text entered in is a number then we need to compare the number to the length of the other text. The largest string or biggest number will be displayed in the results area
    Hint:You will need 4 comparisions (number vs number, string vs string, number vs string, string vs number) and a way to determine if the text is a number.
