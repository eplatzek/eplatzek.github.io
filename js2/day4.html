Day 4: Objects
  • Objects
    • What are objects
    • Vars and Methods:
      JavaScript objects are containers for variables and functions.
      The variables inside of an object are called properties.
      The inside of a object are called methods.
      The properties are written as name:value pairs (name and value separated by a colon).
    • Examples: var car = {make:"Honda", model:"Civic", color:"white", year:2012}; //We can also add methods to an object var car = {make:"Honda", model:"Civic", color:"white", year:2012, description:function() {return "I'm a description!!";}} //We can call that method car.description() //We can also add methods to an object and use the ".this" to reference properties inside of our object var car = {make:"Honda", model:"Civic", color:"white", year:2012, description:function() {return this.make + " " + this.model;}}
      //Let's say we have many objects. We can create a "constructor" to make new objects. //We are using the "this." notation to create variables.     Coding Challenge
    We want to create an address book which will be filled with contacts.
  1. First Challenge: Create html page with input fields to create new contact (first name, last name, phone, etc.)
  2. Second Challenge: Create a button to “save” your contact you just created. Hint: You will need an array to store your created "contacts" each button press should create a new contact and add it to that array.
  3. Third Challenge: Create a button to that displays a list of all of your contacts using DOM manipulation.
  4. Fourth Challenge: Create a search bar to search for contacts and display the ones that match. Hint: You will need to iterate through the array of "contacts" and do a string compare on the matching field/field(s)
