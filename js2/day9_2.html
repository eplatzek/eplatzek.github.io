    • Basics: Two way data binding: <!doctype html> <html ng-app> <head> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.3.14/angular.min.js"></script> </head> <body> <div> <label>Say Hello:</label> <input type="text" ng-model="myStuff.stuff" placeholder="World"> <h1>Hello {{myStuff.stuff}}!</h1> </div> </body> </html> Note if you don’t have Batarang installed go here: https://chrome.google.com/webstore/detail/angularjs-batarang-stable/niopocochgahfkiccpjmmpchncjoapek Note you should use only the “Stable” version. Googling will produce other versions. View the page on your local browser and open inspector. View the AngularJS tab (by installing Baterang if you didn’t already) Alternative references to our data: <h1 ng-bind="myStuff.stuff"> <h1 ng-bind="'Hello ' + myStuff.stuff + '!'"> <h1 ng-bind-html="myStuff.stuff">
      In Angular, we use modules to encapsulate code which allows for re-usability. How to create the module: angular.module(‘UnicornModule’, []); How to start the module in your app: <html ng-app="UnicornModule’"> <!DOCTYPE html> <html> <script src="http://ajax.googleapis.com/ajax/libs/angularjs/1.3.14/angular.min.js"></script> <body> <div ng-app="UnicornModule" ng-controller="kittenController"> {{ firstStuff + " " + secondStuff }} </div> <script> var app = angular.module("UnicornModule", []); app.controller("kittenController", function($scope) { $scope.firstStuff = "Hello"; $scope.secondStuff = "World"; }); </script> </body> </html>
      Here is a Controller: $scope.people = [ {first: 'Joan', last: 'Smith'}, {first: 'John', last: 'Doe'} ]; Here is a simple View: <ul> <li ng-repeat="person in people | orderBy:’last’"> {{ person.first }} {{ person.first }} </li> </ul> Let’s add these to our current controller that has “stuff”.
