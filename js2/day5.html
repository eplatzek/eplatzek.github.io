  • Set up MongoLab
    1. Setup account at: https://mongolab.com/signup/
    2. Verify Email:(wait 10-15 minutes)
    3. Create a new sandbox plan from (https://mongolab.com/plans/)
    4. Click Sandbox again. Create sandbox
    5. Choose Amazon Web Services
    6. Choose a db name (testepdb)
    7. Verify that your new MongoDB Deployment has been deployed.
    8. Click on your new database
    9. Create a new user to connect
  • Download MongoDB software
    1. Go to https://www.mongodb.org/downloads
    2. Download the correct operating system installation
    3. Choose complete
    4. Wait for install to complete.
  • Access Your MongoLab installation through MongoDB
    1. Open your command prompt.
    2. Got to the location where you have mongodb installed.
    3. Copy and paste the code from your mongolab
    4. mongo ds049198.mongolab.com:49198/testepdb -u DBUSER -p PASSWORD
  • Install MongoDB
    1. Install from this site: https://nodejs.org/
    2. Install mongodb by using node packet manager: npm install mongodb
    3. Copy the node example from here: https://github.com/mongolab/mongodb-driver-examples/blob/master/nodejs/nodeSimpleExample.js
    4. Update with your URI info
    5. Run file using: node nodeSimpleExample.js
    6. Comment out Lines 85-91 (this code deletes the tables) and run it again
    7. Check your mongoLab Collections
